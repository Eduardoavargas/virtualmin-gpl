Send automatic reply
When this option is checked, any email to this user will trigger an automatic reply containing the text entered into the adjacent text box. The text can include additional email headers at the top, like : 
Subject: Your autoreply subject
Cc: foo@bar.com

This is your autoreply message.
Have a nice day!
To send an HTML autoreply, start the message with a <html> tag.

Content from the original message can be included in the reply using variables like $SUBJECT, $TO and $FROM.
